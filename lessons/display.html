Display

In this lesson I learned about how to use the display rule here are the types and what they do

Static

The static attribute treats the element(s) like its stuck in place. No rule to move the element will work.

Relative

The relative attribute treats teh element(s)

Absolute

The absolute attribute treats the element(s) like it never existed

TBF

None

None does what it does it doesn't display.

Example:

HTML
            
    <html>
    <head>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <div class="html-ex">
            <p>Hi I'm here</p>
            <p id="hide">Hi you can't see me</p>
        </div>
    </body>
    </html>
CSS: style.css
    .html-ex #hide {
        display: none;
    }
