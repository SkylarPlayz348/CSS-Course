Colors

There are multiple ways to add color to a webpage like text color background color or both

Here are a few examples


    p {color: red;}

    .mainbackground {background-color: green;}

    #loginbtn {color: white; background-color: blue;}

In the example above the first one selects all elements of that type, the second selects all elements with that class, the last on selects the element with that id
